Getting Started with Arduino IDE 2.0

"Arduino is the most popular microController prototyping board used by the hobbyists, makers, and engineers. Arduino is famous for its easy programming, rapid prototyping, pre-built libraries, and tons of support material. In this tutorial we will show you how to get started with Arduino IDE 2.0, ESP32/ESP8266 board packages, serial monitor, and serial plotter.

Downloading and installing Arduino IDE 2.0

The Arduino IDE 2.0 is the enhanced version of the legacy Arduino IDE 1.x that comes with upgraded UI, debugger, board manager, library manager, and auto-complete features.

The Arduino IDE 2.0 can be downloaded from https://www.arduino.cc/en/software.

Operating system requirements for Arduino IDE 2.0 are:

  • Windows – Windows 10 or newer (64 bits)

  • Linux – 64 bits

  • Mac OS X – Version 10.14 or newer (64 bits)

    • From the software download page, select the appropriate Arduino IDE 2.0 version according to your operating system.

After downloading the installation file, simply run it to start the installation process.

Windows Installation

Accept the license agreement to proceed with the installation.

Next up select the installation location.
Wait till the installation process gets completed.
Upon completion of installation process run the Arduino IDE 2.0.

Mac OS Installation

The installation of Arduino IDE 2.0 on Mac OS is quite simple. You just need to copy the installation file into your application folder.

Linux Installation

To install Arduino IDE 2.0 on Linux, first download the AppImage 64 bits from the software download page. After that, we have to convert this downloaded file into an executable file. The process goes as follows:

  • Right click on the downloaded file

  • Select Properties

  • Properties Permissions tab

  • Check the Allow executing file as program box

    • The Arduino IDE 2.0 can then be launched by double clicking the executable file.

How to upload a sketch with Arduino IDE 2.0

The programs written in Arduino IDE are referred to as sketches. Before uploading a sketch to the board, we need to select the appropriate board, set the right COM port, and verify the sketch for errors.

Selecting the board

The steps for selecting the appropriate board in Arduino IDE 2.0 are:

  • Select Tools from the navigation bar

  • Select Port menu option

  • Within Port sub-menu you should be able to see your board name along with COM port. Select the board from here.

Verifying and uploading the sketch

Right below the navigation bar, we can see the tick and arrow icons. The tick icon is used for verifying the sketch while arrow icon is used for uploading the sketch to the board.

When the tick icon is clicked, the sketch gets verified and the results are shown in the black console screen at the bottom. If the code is error-free and compiles successfully then we can proceed to upload the code. By hitting the arrow icon, we can upload the code to the board.

When the sketch uploading is completed successfully, we get notified through the console window at the bottom.

Installing ESP32 and ESP8266 board packages

To install the ESP32 and ESP8266 board packages in Arduino IDE 2.0, go to File Prerferences.

Next we copy the following link in the Additional Board Manager URLs:

http://arduino.esp8266.com/stable/package_esp8266com_index.json,https://dl.espressif.com/dl/package_esp32_index.json

After pasting the URL, we open the Board Manager through Tools  Board  Board Manager.

Search for ESP32/ESP8266 in the search bar and hit install button for ESP32 by Espressif Systems.

After the installation of board package we can select the ESP32/ESP8266 board from the board manager.

After selecting the correct board and COM port, we can upload a sample sketch and test the board.

Installing libraries using library manager

Libraries consist of pre-written code that can be used by the programmers to speed-up their projects. Arduino is known for its extensive list of libraries for different types of peripherals and devices. In this section we will learn how to add libraries in Arduino IDE 2.0 using library manager.

The library manager can be opened through Tools  Manage Libraries.
After opening Library Manager, simply search for the desired library.
Simply click on Install and the desired library will be installed in the Arduino IDE 2.0.

Using Serial Monitor and Serial Plotter in Arduino IDE 2.0

Serial monitor is a great debugging tool available in Arduino IDE. Arduino IDE 2.0 consists of an integrated serial monitor without the need for external windows.

After compiling and uploading the code to the Arduino board, serial monitor can be opened through the Serial Monitor button located at the top right corner. If the serial communication is setup right, the messages will start appearing on the serial monitor window.

Another useful debugging and visualization tool available in the Arduino IDE is serial plotter. Serial plotter visualizes the data obtained from the Arduino boards in a graphical format.

Serial plotter is opened by clicking the serial plotter button in the top right corner of the IDE.
