How to Use ARDUINO IOT Cloud

IOT or internet of things is one of the key enabling technologies for the upcoming fourth industrial revolution a.k.a Industry 4.0. As the name suggest, IOT is a cloud based network where devices send/receive data and communicate with each other. Foreseeing the potential of IOT technology, multiple companies have ventured into IOT platform development business. ARDUINO being the largest player in the open-source embedded hardware/software sector has also jumped the bandwagon. ARDUINO IOT Cloud is an IOT platform that provides a one-stop solution to engineers and makers for creating full-scale IOT projects. Having a project connected to the Arduino IoT Cloud opens up many possibilities, such as tracking data in real time, triggering remote devices and building wireless systems. In this article, we will explore the features and functions of ARDUINO IOT Cloud in detail.

Creating ARDUINO IOT Cloud Account

First of all go to the Arduino IOT Cloud official website: https://cloud.arduino.cc/

If you are new user then you need to register on the platform using the Get Started button on the top right hand corner.

After clicking on the Get Started button, you will be directed to the SignUp page. From here you can either Sign In or create a new account.

After that you will be prompted to enter your date of birth. So enter your DOB and press
Next

After that you will have to select your username, password, and email. You will also be required to agree with the terms and conditions.

After filling all the required fields, just press the Sign Up button. Upon successful sign up, you will be redirected to the ARDUINO Cloud dashboard.

Creating a Thing in ARDUINO IOT Cloud

From the drop down menu on the dashboard, select IOT Cloud.

From the new screen that opens up, press the Create Thing button.

Creating Variables in ARDUINO IOT Cloud

Next up, enter the name of the thing and create variables for the thing.

From the Add Variable window that opens up, we configure the following parameters

  • Name
  • Data Type
  • Variable Permission
  • Variable Update Policy
  • Threshold

After configuring all the parameters, press Add Variable button. All the created variables show up in the dashboard.

Configuring the Device in ARDUINO IOT Cloud

After declaring the variables, you can proceed to configure the device.

Select the device type that you want to configure for IOT Cloud.

You need to download the ARDUINO Create Agent in order to establish communication between ARDUINO device and ARDUINO IOT Cloud platform.

Creating Dashboard in ARDUINO IOT Cloud

After configuring the things, variables, network, and devices, we can proceed to create dashboard. For this, you need to click on the Dashboard tab.

Click on the Build Dashboard button to build a new dashboard.

Edit the name of the new dashboard and place elements on the canvas using Add button.

Using the Add button, we can place widgets as well as things on the dashboard canvas.

We can also have several things running at once, depending on your Arduino IoT Cloud plan, which we can include in the same dashboard. This is a great feature for tracking multiple

boards in for example a larger sensor network, where boards can be connected to different networks around the world, but be monitored from the same dashboard.
